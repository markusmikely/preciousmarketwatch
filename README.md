# Precious Market Watch

A modern, AI-powered WordPress affiliate blog providing insights about precious metals and gemstones throughout their lifecycle from mining to storage.

## 🚀 Features
- **AI-Powered Content Generation**: Automated research and article creation
- **Real-time Market Data**: Live precious metals prices and trends
- **Modern React UI**: Fast, responsive user interface
- **Headless WordPress**: Content management via GraphQL API
- **Multi-platform Publishing**: Automated social media and marketing content

## 📁 Project Structure

## 📁 Project Structure

```text
├── frontend/ # React.js application
├── backend/ # Headless WordPress setup
├── pmw-agents/ # Python LangChain/LangGraph agents
├── database/ # PostgreSQL schemas and migrations
├── deployment/ # Docker, CI/CD configurations
└── docs/ # Project documentation
```

## 🛠️ Tech Stack
- **Frontend**: React 18, TypeScript, Tailwind CSS, React Router
- **Backend**: WordPress (Headless), GraphQL, Custom PHP plugins
- **AI Engine**: Python 3.11, LangChain, LangGraph, OpenAI GPT-4
- **Database**: PostgreSQL 15, Redis for caching
- **Deployment**: Docker, GitHub Actions, AWS EC2/RDS
